== README

* Ruby version => 2.1.0

* Rails version => 4.0.2 

* Run <tt>rake setup</tt> to create the database, run migrations, create dummy data.

* How to run the test suite => <tt>rake test</tt>

* Note: All users created are given a password 'welcome123' by default.

* Assumptions: A customer need not have an account to view the appointments.

* Note: A link to scheduled appointments is shown on login page for customers to view all appointments.
